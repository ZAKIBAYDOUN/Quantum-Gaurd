## 📦 Quick Start

### Prerequisites
- Node.js 20+ 
- PostgreSQL database
- Python 3.11+ (for blockchain core)

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/quantum-guard-wallet.git
cd quantum-guard-wallet

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env
# Edit .env with your database URL and secrets

# Start development server
npm run dev
```

### Environment Variables

```env
DATABASE_URL=postgresql://username:password@localhost:5432/quantum_guard
SESSION_SECRET=your-secure-session-secret
NODE_ENV=development
```
